# Student Enrollment Form

## Description

This project is a Student Enrollment Form developed using HTML, Bootstrap, JavaScript, and JsonPowerDB (JPDB).

The form stores student data into the STUDENT-TABLE relation of SCHOOL-DB database.

The project performs:

- Save student data
- Update existing student records
- Reset form
- Validate input fields
- Search records using Roll Number

Primary Key:
- Roll No

Input Fields:
- Roll No
- Full Name
- Class
- Birth Date
- Address
- Enrollment Date
